Projects Highlighted

Cloud Resume Challenge

I built and deployed a full-stack serverless web application using Azure services. The frontend is hosted on Azure Blob Storage with static website hosting and integrated with Azure Front Door for global content delivery and custom domain management. The backend is powered by Azure Functions using the Node.js v4 model, connected to Azure Cosmos DB to track and store visitor counts. I implemented CI/CD pipelines through GitHub Actions for both frontend and backend deployments, and configured CORS, DNS, and custom domain validation. The project also included setting up budget alerts and monitoring through Azure Cost Management.

Splunk-Based Data Analysis

This GitHub-hosted Splunk project showcases data analysis and Visualization of U.S. Police Shootings using a dataset from Kaggle. It includes the use of event types and tags for better data organization, lookup tables to enhance the data, and calculated fields to extract insights. The project also utilizes basic and parameterized macros to make search queries more efficient. It features interactive reports and dashboards that provide clear visualizations and summaries.
